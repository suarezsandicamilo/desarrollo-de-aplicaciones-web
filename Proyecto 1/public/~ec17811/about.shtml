Acerca De

Este sitio web es un proyecto que forma parte de mi curso de Desarrollo de Aplicaciones Web. Su propósito principal es proporcionar una plataforma donde puedo aplicar y demostrar mis conocimientos en HTML, CSS y JavaScript, así como explorar tecnologías adicionales como Bootstrap y Apache HTTP Server con Server Side Includes (SSI).

Aprendizaje y Desarrollo

Como estudiante apasionado de Computación, valoro la oportunidad de aprender a través de la práctica y la aplicación de mis habilidades. Este proyecto me ha permitido adentrarme en el mundo del desarrollo web y adquirir experiencia práctica en la creación de un sitio web interactivo y funcional.

Tecnologías Utilizadas

  • HTML: HTML es el lenguaje fundamental que utilizo para estructurar y organizar el contenido de este sitio web.
  • CSS: CSS se emplea para dar estilo y diseño al sitio web, asegurando una presentación atractiva y coherente.
  • JavaScript: JavaScript se utiliza para la interactividad y la funcionalidad dinámica del sitio web. Esto permite que la página responda a las acciones del usuario de manera eficiente.
  • Bootstrap: Bootstrap es un marco de diseño de código abierto que simplifica el proceso de diseño web al proporcionar componentes y estilos predefinidos que mejoran la apariencia y la usabilidad del sitio.
  • Apache HTTP Server con Server Side Includes: Esta configuración de servidor permite la inclusión de fragmentos de código HTML en las páginas web a través de SSI. Esto mejora la eficiencia y la facilidad de mantenimiento del sitio.
